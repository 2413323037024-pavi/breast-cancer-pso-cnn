# breast-cancer-pso-cnn
# Breast Cancer Detection Using CNN and PSO Optimization

## Overview

This project focuses on detecting breast cancer using Convolutional Neural Networks (CNN) integrated with Particle Swarm Optimization (PSO) for hyperparameter tuning. The system improves classification accuracy and provides real-time prediction through a Streamlit web application.

## Features

* Breast cancer classification using CNN
* Hyperparameter optimization using PSO
* Real-time prediction using Streamlit
* User-friendly interface
* Performance evaluation and visualization

## Technologies Used

* Python
* TensorFlow / Keras
* Scikit-learn
* Streamlit
* NumPy
* Pandas
* Matplotlib

## Dataset

Wisconsin Breast Cancer Dataset

## Project Workflow

1. Data Preprocessing
2. CNN Model Development
3. PSO Hyperparameter Optimization
4. Model Training and Testing
5. Performance Evaluation
6. Streamlit Deployment
